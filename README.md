# Naim Badawi - Portfolio Website

A modern, responsive portfolio website showcasing professional experience, projects, and skills in software engineering and cybersecurity.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Interactive Navigation**: Smooth scrolling and mobile-friendly menu
- **Sections Included**:
  - Hero section with call-to-action buttons
  - About section with statistics
  - Technical skills categorized by type
  - Professional experience with detailed descriptions
  - Featured projects showcase
  - Education information
  - Contact section with social links

## File Structure

```
├── index.html    # Main HTML file
├── styles.css    # All styling and responsive design
├── script.js     # Interactive functionality
└── README.md     # This file
```

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (ES6+)
- Responsive Design Patterns

## How to Use

1. Open `index.html` in your web browser
2. Navigate through sections using the navigation menu
3. Click on buttons to contact or view projects
4. Social media links in the hero and contact sections direct to external profiles

## Customization

### Update Resume Information
- Edit the content in `index.html` to match your current information
- Update dates, company names, and descriptions as needed

### Change Colors
Modify the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --text-dark: #1a1a1a;
    --bg-light: #f8f9fa;
    /* ... other colors ... */
}
```

### Add New Sections
1. Add HTML structure in `index.html`
2. Style with CSS in `styles.css`
3. Add JavaScript interactions in `script.js` if needed
